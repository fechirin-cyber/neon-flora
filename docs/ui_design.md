# NEON FLORA UI デザイン仕様

## 1. フォント選定リスト

| 用途 | フォント候補 | 備考 |
|---|---|---|
| **メインUI（CREDIT/BIG/REG）** | DSEG7 Classic | 7セグメント風。パチスロのデータカウンター準拠。α導入予定 |
| **図柄名・ラベル** | Noto Sans JP Bold | 日本語対応。視認性高い |
| **演出テキスト** | M PLUS Rounded 1c | 丸ゴシック。柔らかい印象。VTuber世界観に合致 |
| **デバッグ情報** | JetBrains Mono | 等幅。デバッグオーバーレイ用 |
| **フォールバック** | Godot Default (Noto) | 組み込みフォント。プリプロで使用中 |

### フォントサイズ階層

| レベル | サイズ | 用途 |
|---|---|---|
| H1 | 32px | ゲーム状態表示（State, BONUS等） |
| H2 | 28px | クレジット、統計情報 |
| Body | 24px | 演出情報、リーチ目告知 |
| Button | 22px | ボタンラベル |
| Caption | 18px | ヘルプテキスト、デバッグ |

## 2. 筐体レイアウト 7ゾーニング

画面サイズ: 900 x 1600 px（縦長、portrait）

```
┌─────────────────────────┐  Y=0
│  Zone 1: ステータスバー   │  40-90px (50px)
│  State / G数 / BIG / REG │
├─────────────────────────┤  Y=100
│  Zone 2: データカウンター  │  100-180px (80px)
│  CREDIT / IN / OUT        │
├─────────────────────────┤  Y=200
│                           │
│  Zone 3: リール表示窓     │  200-560px (360px)
│  3x3図柄 + 5ライン表示    │
│  各図柄: 200x160px        │
│                           │
├─────────────────────────┤  Y=580
│  Zone 4: 演出情報エリア   │  580-740px (160px)
│  フラグ / 消灯 / リーチ目  │
│  入賞演出 / WIN表示        │
├─────────────────────────┤  Y=760
│  Zone 5: ボタンパネル      │  760-980px (220px)
│  BET / LEVER / STOP L/C/R │
│  最小タッチターゲット: 48dp │
├─────────────────────────┤  Y=1000
│                           │
│  Zone 6: キャラクター表示  │  1000-1400px (400px)
│  ひかり / るな / こはる     │
│  リアクション表示エリア     │
│                           │
├─────────────────────────┤  Y=1420
│  Zone 7: ヘルプ・デバッグ  │  1420-1580px (160px)
│  操作説明 / デバッグコマンド │
└─────────────────────────┘  Y=1600
```

### ゾーン詳細

| Zone | 名称 | Y範囲 | 高さ | 内容 |
|---|---|---|---|---|
| 1 | ステータスバー | 40-90 | 50px | GameState, 総ゲーム数, BIG/REG回数 |
| 2 | データカウンター | 100-180 | 80px | CREDIT, IN, OUT, ボーナス中残G/払出 |
| 3 | リール表示窓 | 200-560 | 360px | 3リール×3段、各図柄200×160px、隙間4px |
| 4 | 演出情報 | 580-740 | 160px | フラグ表示、消灯レベル、リーチ目、WIN演出 |
| 5 | ボタンパネル | 760-980 | 220px | BET/LEVER/STOP×3、タッチターゲット48dp以上 |
| 6 | キャラクター | 1000-1400 | 400px | VTuberキャラリアクション表示 |
| 7 | ヘルプ | 1420-1580 | 160px | 操作説明、デバッグ情報 |
